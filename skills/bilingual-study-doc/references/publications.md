# Publication Patterns Reference

## Data Sources

### awesome-english-ebooks (Primary)

**Repository**: https://github.com/hehonghui/awesome-english-ebooks

Weekly/monthly updated English periodicals in EPUB, MOBI, PDF formats.

| Directory | Publication | Update Frequency |
|-----------|------------|------------------|
| `01_economist` | The Economist | Weekly (Friday 23:00) |
| `02_new_yorker` | The New Yorker | Weekly (Saturday AM) |
| `04_atlantic` | The Atlantic | Monthly (2nd) |
| `05_wired` | Wired | Monthly (2nd) |

**Usage**: Clone the repo or download individual issues. PDF format recommended for text extraction.
